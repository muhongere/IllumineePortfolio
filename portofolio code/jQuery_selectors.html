The user has to add a selector to add the class ‘odd’ to the odd number rows (e.g. rows 1,3, and 5) • Add a selector to add the class ‘even’ to the even numbered rows (e.g. rows 2, 4, and 6) • Add a selector to add the class ‘highlighter’ to the 4th item in the list • Add a selector to add the class ‘last’ to the last item in the list
